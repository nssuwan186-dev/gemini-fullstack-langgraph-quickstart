# เทคโนโลยีที่ใช้ (Tech Stack)

โปรเจกต์นี้เป็นแอปพลิเคชัน Fullstack ที่ใช้เทคโนโลยีหลากหลายเพื่อสร้าง Agent วิจัยที่ทรงพลังและอินเทอร์เฟซผู้ใช้ที่ทันสมัย

## 1. ส่วนหน้า (Frontend)
*   **React:** ไลบรารี JavaScript ยอดนิยมสำหรับการสร้าง User Interface (UI) แบบโต้ตอบ
*   **Vite:** เครื่องมือ build ที่รวดเร็วและน้ำหนักเบาสำหรับโปรเจกต์ JavaScript สมัยใหม่
*   **Tailwind CSS:** เฟรมเวิร์ก CSS แบบ utility-first สำหรับการสร้างดีไซน์ที่กำหนดเองได้อย่างรวดเร็ว
*   **Shadcn UI:** ชุดคอมโพเนนต์ UI ที่สร้างขึ้นบน Tailwind CSS เพื่อความสวยงามและใช้งานง่าย

## 2. ส่วนหลัง (Backend)
*   **Python:** ภาษาโปรแกรมหลักสำหรับตรรกะฝั่งเซิร์ฟเวอร์และการทำงานของ Agent
*   **LangGraph:** เฟรมเวิร์กสำหรับการสร้างแอปพลิเคชันหลาย Agent ที่มีความยืดหยุ่นและมีสถานะ (stateful)
*   **FastAPI:** เว็บเฟรมเวิร์ก Python ที่รวดเร็ว (high-performance) สำหรับการสร้าง API

## 3. โมเดลและบริการ AI
*   **Google Gemini Models:** โมเดลภาษาขนาดใหญ่ (LLMs) ของ Google ใช้สำหรับการสร้างคำค้นหา การไตร่ตรอง และการสังเคราะห์คำตอบ
*   **Google Search API:** ใช้สำหรับทำการวิจัยบนเว็บและดึงข้อมูลจากอินเทอร์เน็ต

## 4. ฐานข้อมูลและแคช (สำหรับ Production)
*   **Redis:** ใช้เป็น pub-sub broker สำหรับการสตรีมเอาต์พุตแบบเรียลไทม์ และสำหรับการจัดการคิวงานเบื้องหลัง
*   **Postgres:** ใช้สำหรับจัดเก็บ assistants, threads, runs, สถานะของ thread และหน่วยความจำระยะยาว
