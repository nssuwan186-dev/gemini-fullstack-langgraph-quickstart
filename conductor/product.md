# แนวคิดเริ่มต้น
แอปพลิเคชัน Fullstack นี้ใช้ React สำหรับส่วนหน้าและ LangGraph Agent สำหรับส่วนหลัง Agent นี้ได้รับการออกแบบมาเพื่อทำการวิจัยอย่างครอบคลุมเกี่ยวกับคำถามของผู้ใช้โดยการสร้างคำค้นหาแบบไดนามิก การสอบถามเว็บโดยใช้ Google Search การไตร่ตรองผลลัพธ์เพื่อระบุช่องว่างความรู้ และการปรับปรุงการค้นหาซ้ำๆ จนกว่าจะสามารถให้คำตอบที่สนับสนุนด้วยการอ้างอิงได้ แอปพลิเคชันนี้ทำหน้าที่เป็นตัวอย่างของการสร้าง AI สนทนาที่เสริมด้วยการวิจัยโดยใช้ LangGraph และโมเดล Gemini ของ Google

## คุณสมบัติหลัก

*   แอปพลิเคชัน Fullstack พร้อมส่วนหน้า React และส่วนหลัง LangGraph
*   ขับเคลื่อนโดย LangGraph Agent สำหรับการวิจัยขั้นสูงและ AI สนทนา
*   การสร้างคำค้นหาแบบไดนามิกโดยใช้โมเดล Google Gemini
*   การวิจัยเว็บแบบบูรณาการผ่าน Google Search API
*   การใช้เหตุผลแบบไตร่ตรองเพื่อระบุช่องว่างความรู้และปรับปรุงการค้นหา
*   สร้างคำตอบพร้อมการอ้างอิงจากแหล่งข้อมูลที่รวบรวมได้
*   Hot-reloading สำหรับทั้งส่วนหน้าและส่วนหลังระหว่างการพัฒนา
